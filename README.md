# 👗 Royalty Fabric Website

## 📌 Project Overview

Royalty Fabric is a responsive fashion and textile website developed using HTML, CSS, and JavaScript. The website showcases premium fabrics, bespoke tailoring services, and provides a modern user experience with interactive features and responsive design.

This project was created as part of a web development assignment focusing on responsive design, interactivity, SEO optimisation, and external service integration.

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Maps Embed
- WhatsApp API (Click-to-chat)

---

## 🎯 Features

### 🌐 Responsive Design
- Fully responsive layout for desktop, tablet, and mobile devices
- Mobile-friendly hamburger navigation menu

### ⚙️ JavaScript Functionality
- Form validation (name, email, message)
- Scroll-based animations
- Interactive hamburger menu

### 🔍 SEO Optimisation
- Meta titles and descriptions
- Keywords for search engines
- Image alt text improvements
- Mobile viewport settings

### 📩 Functional Contact Form
- Client-side validation
- Dropdown subject selection
- User agreement checkbox

### 🌍 External Integrations
- Google Maps embedded location (Rosebank College Braamfontein Campus)
- WhatsApp floating chat button for direct communication
